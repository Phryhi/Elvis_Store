# Elvis Store - Lista de Emails de Clientes

Este é um projeto back-end simples (sem estilização) desenvolvido em **PHP** com banco de dados **MySQL**, que permite o cadastro de clientes interessados em receber novidades da loja **Elvis Store - MakeMeElvis**.

## Funcionalidades

- Cadastro de clientes com **nome**, **sobrenome** e **email**
- Armazenamento das informações em um banco de dados **MySQL**
- Envio de emails automáticos para todos os clientes cadastrados
- Exclusão de clientes específicos da lista

## Tecnologias utilizadas

- PHP
- MySQL
- HTML

## Ambientes de Desenvolvimento

- XAMPP
- VSCode
